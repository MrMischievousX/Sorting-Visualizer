# Sorting Visualizer

## This is a simple Visualization project made using Javascript

- Bubble Sort
- Selection Sort
- Quick Sort
- Merge Sort

You can also change the size of Array and also change the speed of Sorting.

## This is built only using HTML, CSS, JavaScript
